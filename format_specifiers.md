# C 言語フォーマット指定子

## 基本的な型指定子
- `%d` - int (10進数)
- `%u` - unsigned int
- `%ld` - long
- `%lld` - long long
- `%f` - float/double
- `%lf` - double (scanf)
- `%c` - char
- `%s` - 文字列
- `%p` - ポインタ
- `%x` - 16進数 (小文字)
- `%X` - 16進数 (大文字)
- `%o` - 8進数

## 幅とパディング
- `%5d` - 最小5文字幅 (右詰め)
- `%-5d` - 最小5文字幅 (左詰め)
- `%05d` - 0で埋める (例: 00042)
- `%*d` - 幅を引数で指定

## 小数点
- `%.2f` - 小数点以下2桁
- `%7.2f` - 全体7文字、小数点以下2桁
- `%.0f` - 小数点以下なし
- `%g` - 不要な0を省略

## 文字列
- `%.10s` - 最大10文字
- `%20s` - 最小20文字幅
- `%-20.10s` - 左詰め、最小20文字、最大10文字

## 特殊
- `%%` - % 記号
- `%n` - これまでの出力文字数を格納
- `%e` - 指数表記 (3.14e+00)
- `%E` - 指数表記 (3.14E+00)

## よく使う例
```c
printf("%02d", 5);        // 05
printf("%7.2f", 3.14159); // "   3.14"
printf("%-5s", "Hi");     // "Hi   "
printf("%#x", 255);       // 0xff
```
